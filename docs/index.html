Notes From the Field, ALA Booklist - Article here!

A short piece I wrote for Booklist detailing my work at the Indianapolis Public Library to create our first LGBTQ+ committee, with the aims of improving programming, collections, and staff development aimed at the LGBTQ+ community.

Queerbrary - Queerbrary.com

My pet project, which I frankly need to devote more time to. Queerbrary is a place to talk all things LGBTQ+ librarianship: outreach basics, improving collections, programming ideas, book reviews, etc.

Library Journal Reviews - libraryjournal.com

I'm constantly reviewing something new for Library Journal. One of the benefits of my unique, hard-to-spell name is that it's a guarantee that my last name won't refer to anyone else's work, so I can grab it all for you in one convenient search! Some of my favorite books I've reviewed to date for LJ include The Anthropocene Reviewed by John Green, Unspeakable Acts by Sarah Weinman, and Fat and Queer, edited by Jessica Kingsley.
